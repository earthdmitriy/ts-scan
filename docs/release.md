Release checklist:

1. Run tests with `npm run test`
2. Build with `npm run build`
3. Install locally with `npm run install-local`
4. Run local version in project `npx ts-scan ...`
5. Uninstall local version `npm run remove-local`
6. Bump version `npm version patch` or `npm version minor`
7. Update readme
8. Push changes
9. Run `npm publish`
